# 初期构建指令

首先安装 node

安装 Create React App 工具（如果尚未安装）：

```bash
npm install -g create-react-app
```

ts 版本：

```bash
npx create-react-app my-app --template typescript
```

使用 vite 创建

```bash
npm create vite@latest my-react-app -- --template react
```

创建新的 React 应用：

```bash
create-react-app my-app
```
